# 📄 Text Similarity Detection using LSTM

## 🔹 Objective

To build a system that detects similarity between two text documents using a neural network (LSTM).

---

## 🔹 Methodology

* Data preprocessing (tokenization & padding)
* Convert text into sequences
* Use Siamese LSTM model
* Compare outputs using dense layer
* Generate similarity score (0–1)

---

## 🔹 Technologies Used

* Python
* TensorFlow / Keras
* Streamlit
* NumPy, Pandas

---

## 🔹 How to Run

### Step 1: Install dependencies

pip install tensorflow numpy pandas scikit-learn streamlit

### Step 2: Train model

python model.py

### Step 3: Run application

streamlit run app.py

---

## 🔹 Output

* User enters two texts
* Model predicts similarity score
* Displays whether texts are similar or not

---

## 🔹 Applications

* Plagiarism detection
* Chatbots
* Search engines
* Recommendation systems

---

## 🔹 Conclusion

This project demonstrates how LSTM-based neural networks can effectively measure semantic similarity between texts.
